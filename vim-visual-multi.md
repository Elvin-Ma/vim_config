# 1 多光标操作插件

## 1.1 install
**vim-plug**
```shell
Plug 'mg979/vim-visual-multi', {'branch': 'master'}
```

**vim8+**
```shell
mkdir -p ~/.vim/pack/plugins/start && git clone https://github.com/mg979/vim-visual-multi ~/.vim/pack/plugins/start/vim-visual-multi
```

## 1.2 use guide

**document**
```vim
:help visual-multi
```

**look tutorial**
```shell
vim -Nu path/to/visual-multi/tutorialrc
```
